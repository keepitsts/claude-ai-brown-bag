### What's in the Repo

The repo is a complete record of everything in this presentation:

- `prompts/` — Every prompt given to Claude, numbered sequentially (001–017)
- `research/` — All research artifacts (STS briefing, API list, demo spec, tech stack, build strategy)
- `build/prompts/` — The 11 headless build prompts (00-scaffold through 10-integration)
- `build/build.sh` — The orchestration script with verification gates
- `trailhead/` — The complete application source code (4,455 lines, 58 files)
- `trailhead/CLAUDE.md` — The conventions file that guided every build step
- `presentation/` — The slides and research for this presentation

The entire process is transparent and reproducible.
